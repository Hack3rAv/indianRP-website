ADMIN RANKS

Administrative hierarchy and role definitions. Authority is strictly enforced based on rank.

RANK STRUCTURE

SERVER MANAGER

Highest authority. Oversees the entire server, administration team, and final decisions.

APPEAL ADMIN

Handles punishment appeals and reviews administrative actions for fairness.

TECHNICAL ADMIN

Responsible for server stability, systems, and technical operations.

SUPPORT ADMIN

Manages player support, tickets, and community assistance.

ADMIN INTERN

Entry-level administrative role under supervision. Limited permissions.
